## DSFB2 Portfiolio
Data Science for Biology 2 Portfolio 

# Contents
* README.md = Main documentation file
* .gitignore = Git ignore rules
* .dsfb2_workflows_portfolio.Rproj = Rstudio project file
* markdown/ = Folder with all the .Rmd pages for the book and supporting files
 * _book/ = output of render_book, used for building the Github page
 * _bookdown.yml = rules for generating the _book
 * 00_index.Rmd = first page of book, required to build Github page
 * XX_... Rmd = .Rmd pages in the book 
 * CV_template/ = contains 20secondscv template to create CV_template_1.pdf
  * CV_template.Rmd = .Rmd CV template
 * .png files = screenshots and CV template .png for showing them in the book
*rna_seq_guerilla/ = rna seq project organisation according to guerilla analytics
