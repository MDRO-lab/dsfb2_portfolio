#RNA-sequencing formatieve opdracht


## Description
Deze folder bevat de folder structuur van de rna-seq formatieve opdracht, 
aangepast volgens Guerilla analytics data management.

## folderstructure
* 01_data/ = bevat bewerkte data
 * bam/ = .bam files 
 * fastqc/ = fastqc files
 * count_table/ = count table  folder
 * alignment/ = alignment_table folder
 * metadata_sample/ = bevat sample metadata
* 02_markdown/ = bevat de markdown opdrachten en volledige versie
 * opdracht_modules/ = alle opdrachten opgedeeld in modules
 * rna_seq_formatieve_opdracht = volledige formatieve opdracht
* 03_raw_data/ = bevat ruwe data
 * fastq/ = bevat fastq data met sample identifiers voor script
* 04_scripts/ = bevat alle R and bash scripts
* README.md = Main documentation file
